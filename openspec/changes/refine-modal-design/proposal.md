# Change: Refine Modal Design

## Why
The Contact and Sponsor modals currently have two critical issues:
1. Text overlap causing readability problems
2. Poor visual design that doesn't match the app's elegant aesthetic

## What Changes
- Fix text overlap issues in both modals
- Redesign modal layout with proper spacing and hierarchy
- Apply consistent visual style matching the main app (stone color palette, serif fonts)
- Improve modal backdrop and animation
- Add proper content structure for Contact modal (email, WeChat QR code placeholder)
- Add proper content structure for Sponsor modal (payment QR code placeholder)

## Impact
- Affected specs: visual-design
- Affected code: 
  - src/components/ContactModal.tsx
  - src/components/SponsorModal.tsx
