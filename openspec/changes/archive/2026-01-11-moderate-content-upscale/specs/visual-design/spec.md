# Visual Design Specs

## MODIFIED Requirements

### Requirement: Global Scale-Up
All content elements MUST be scaled up moderately using inline pixel styles.
#### Scenario: Moderate Sizes
- **Given** the app is rendered.
- **Then** the Clock digits MUST be `56px`.
- **And** the Pillar text MUST be `48px`.
- **And** the Relationship List items MUST be `24px`.
