# Visual Design Specs

## MODIFIED Requirements

### Requirement: Global Scale-Up
All content elements (excluding the main title) MUST be scaled up by one size tier.
#### Scenario: Extra Large Content
- **Given** the app is rendered.
- **Then** the Clock digits MUST be `text-7xl`.
- **And** the Chart Pillars MUST be `text-7xl`.
- **And** the Chart Headers MUST be `text-5xl`.
- **And** the Relationship List items MUST be `text-3xl`.
- **And** the Cloud Icons MUST be `96px` wide.
