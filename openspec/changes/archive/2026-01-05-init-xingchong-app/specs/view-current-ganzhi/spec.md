# Spec: View Current GanZhi

## ADDED Requirements

### Requirement: Display Current Date in Gan-Zhi
The application MUST prominently display the current date and time expressed in the Heavenly Stems and Earthly Branches (Gan-Zhi) system.

#### Scenario: App Load
- Given the user opens the application
- When the page loads
- Then the current Year, Month, Day, and Hour are displayed in Gan-Zhi characters (e.g., "乙巳年 丙xx月...").
