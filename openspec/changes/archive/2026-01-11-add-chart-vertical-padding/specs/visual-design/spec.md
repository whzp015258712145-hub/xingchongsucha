# Visual Design Specs

## ADDED Requirements

### Requirement: Chart Spacing
The chart MUST have sufficient vertical whitespace around the Heaven and Human sections to avoid a cramped appearance.
#### Scenario: Vertical Padding
- **Given** the chart is rendered.
- **Then** there MUST be extra whitespace above the "Heaven" header.
- **And** there MUST be extra whitespace below the "Human" Bazi pillars.
